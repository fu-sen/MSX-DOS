MODE
-----------------------------------------------------------
[ 機能 ]
ディスプレイの画面表示幅(横)を設定します。

[ 書式 ]
MODE <表示文字幅>

[ 文例 ]
MODE 40

[ 解説 ]
<表示文字幅> の範囲は MSX(1) で 1〜40、MSX2 以降で 1〜80 です。1〜32 の場合は BASIC の SCREEN 1 に、33 以上の場合は BASIC の SCREEN 0 に設定されます。
MODE の実行により、ディスプレイの表示は消去されます。

BASIC の SCREEN と WIDTH に該当します。
